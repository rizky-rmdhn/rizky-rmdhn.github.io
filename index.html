03. Project

// Deep Learning & Computer Vision

Sistem Klasifikasi Sampah Organik & Anorganik Berbasis Arsitektur EfficientNet

Mengembangkan model *Deep Learning* menggunakan Framework YOLOv8 dan Arsitektur **EfficientNet** untuk mengklasifikasikan sampah ke dalam kategori organik dan anorganik. Model ini dilatih secara mandiri menggunakan dataset organik dan anorganik yang bersifat organik/asli (bukan hasil cropping), sehingga memiliki generalisasi yang lebih baik saat mendeteksi objek utuh di lingkungan nyata.

Python YOLOv8 EfficientNet PyTorch / TensorFlow Image Classification
// Natural Language Processing (NLP) & Data Science

Analisis Sentimen Komentar YouTube Berbasis NLP

Membangun sistem klasifikasi teks menggunakan teknik *Natural Language Processing* (NLP) untuk menganalisis dan mengelompokkan sentimen komentar pada platform YouTube ke dalam kategori positif dan negatif. Proyek ini melibatkan proses *text preprocessing* yang komprehensif seperti *cleansing, case folding, tokenizing, filtering (stopword removal)*, dan *stemming* sebelum dilakukan ekstraksi fitur dan pemodelan klasifikasi.

Python NLP Text Preprocessing Scikit-Learn / NLTK
